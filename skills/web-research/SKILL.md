# Web Research Skill

Use this skill to collect up-to-date information from the web and organize findings.

## Workflow

1) Clarify the research question and key terms.
2) Run multiple queries on reputable sources; capture URLs and publication dates.
3) Skim for relevance, then read deeply; prefer primary sources and recent updates.
4) Extract key facts, numbers, and quotes with citations.
5) Organize notes into bullet points grouped by subtopic.
6) Synthesize a concise summary with sources listed.

## Tips

- Prefer official docs, standards bodies, and well-known publications.
- Note contradictions and flag uncertainties.
- Record sources as absolute URLs with dates when available.
